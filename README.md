# 2017-VAST-challenge-MC1

- Annual challenge / competition in Visual Analytics community
- In 2017, the task is to identify potential culprits stealing birds at a park
- We were given a park map and visitor log (car type, entry data with time)
- My team represented the park map and visitor data as graph 
- Written in Python, exploring Dash and NetworkX to visualize graph and hypotheses as to “culprit”
- Competition page: https://www.vacommunity.org/VAST+Challenge+2017+MC1
